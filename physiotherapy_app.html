Physiotherapy App

UIUX Designer - Research / Flow / Prototyping

scroll

How might we help elders rehabilitate directly at home?

This project is an App for the company's physical therapy clinic. It is inconvenient for patients to go out mainly in response to the public. Still, this product can do a simple test for the user and give corresponding rehabilitation. Provides guidance and examples of user actions so that users can treated comfortably at home.

Flow

Roles

Researcher / Commuication / Prototype
RD team is the department in which I worked in the company. I discussed the flow with the PM and discovered that usability is the early term I worked on. During this process, I must interview the patients to figure out the actual need. I was then meeting with the physiotherapist to check for the functions.
We discussed the logic of API with back-end engineers to check the functions before starting to create the app's prototypes from the wireframe.

Wireframe

Functions & Advantages

Create an useful app for elders.

  • Create rehabilitation habits for the elders.
    Physiotherapists will customize patient's rehabilitation sets for them to exercise at home. It will set up the exercises, and the app will remind users to start working out on time.
  • Users can self-test for their pain.
    We created a few questions for users to answer. Then, based on the score, the user customizes pain relief poses to do or make an appointment for physiotherapy.
  • Reduce communication costs with physiotherapists
    The app will record the user's every rehabilitation score and send the data to the background system. It is easy for physiotherapists to track the user's rehabilitation state.

The report shows the app helps patients increase 40% of their rehabilitation efficiency. Also, it accelerated the physiotherapist's working process.
